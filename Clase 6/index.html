Eventos

Se definen como acciones que se ejecutan tanto por el navegador como por el usuario.

Modelo básico de eventos

  • Características limitadas
  • Funciona en todos los navegadores
  • Se llaman a través de funciones en línea con las etiquetas HTML

Modelo de eventos estándar

  • Se utiliza para temas complejos
  • Funciona en todos los navegadores
  • Se llaman a través de nuestros scripts de JS
